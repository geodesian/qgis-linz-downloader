# LINZ Data Downloader

A QGIS plugin for downloading geospatial data from LINZ Data Service with polygon-based area selection.

## Features

- Download DEMs, cadastral data, imagery, and vector layers from LINZ Data Service
- Polygon-based area selection using drawing tools or existing layers
- Parallel coverage validation for fast search results
- Automatic raster repair for corrupted downloads
- Integration with Geodesian Tools toolbar
- Automatic clipping to selected area

## Installation

1. Download the plugin
2. Extract to your QGIS plugins directory
3. Enable the plugin in QGIS Plugin Manager
4. Get an API key from [data.linz.govt.nz](https://data.linz.govt.nz/my/api/)

## Usage

1. Click the LINZ Data Downloader button in the Geodesian Tools toolbar
2. Enter your LINZ API key
3. Draw or select an area of interest
4. Search for available datasets
5. Select datasets to download
6. Choose output folder and download
